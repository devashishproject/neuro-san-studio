# HealthHire AI

HealthHire AI is a Healthcare Resume Analyzer built using Neuro SAN.

## Features

- Resume Parsing
- Healthcare Skill Detection
- Job Role Matching
- Skill Gap Analysis
- Career Recommendations

## Use Cases

- Hospitals
- Healthcare Recruiters
- Medical Coders
- Clinical Researchers
- Healthcare Analysts

## Run

```bash
ns run
`
##Technology

Neuro SAN
Python
Mistral/OpenAI/Gemini
